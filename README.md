# Проверщик грамматики
```
 Онлайн-сервис, позволяющий проверять орфографию и стилистику текста. После отправки текста пользователю возвращаются исправления и рекомендации.
 ```

# Реализованные функции:
```
Интуитивный интерфейс для вставки текста.
Подсветка и коррекция ошибок.
Анализ структуры предложений и стилей письма.
Технология:

NLTK библиотека для обработки естественного языка.
Django формы для взаимодействия с пользователями.
Шаблонизатор Django для рендеринга результатов анализа.
```
# Активация виртуального окружения:
